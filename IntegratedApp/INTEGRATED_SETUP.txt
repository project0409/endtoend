## 🚀 SETUP INSTRUCTIONS

### Step 1: Install Dependencies
```bash
cd IntegratedApp\frontend
npm install
```

### Step 2: Start ALL Backend Services (Required!)
You MUST start all 4 backend services before running the frontend:

**Terminal 1 - MongoDB:**
```bash
mongod
```

**Terminal 2 - Module 1 Backend (Port 5001):**
```bash
cd Module1_UserAuthentication\backend
npm install
npm start
```

**Terminal 3 - Module 2 Backend (Port 5002):**
```bash
cd Module2_ExperienceSharing\backend
npm install
npm start
```

**Terminal 4 - Module 3 Backend (Port 5003):**
```bash
cd Module3_PreparationGuidance\backend
npm install
npm start
```

**Terminal 5 - Module 4 Backend (Port 5004):**
```bash
cd Module4_SkillGapAnalysis\backend
npm install
npm start
```

### Step 3: Start Integrated Frontend
**Terminal 6:**
```bash
cd IntegratedApp\frontend
npm start
```

The app will open at: **http://localhost:3000**

## 🐛 TROUBLESHOOTING

### Frontend won't start:
```bash
cd IntegratedApp\frontend
rmdir /s /q node_modules
del package-lock.json
npm install
npm start
```

### Backend connection errors:
- Ensure ALL 4 backends are running
- Check ports: 5001, 5002, 5003, 5004
- Verify MongoDB is running on port 27017

### CORS errors:
- All backends have CORS enabled for http://localhost:3000
- If issues persist, restart all backend services

## 🚀 QUICK START SCRIPT

Create `start-all.bat` in project root:

```batch
@echo off
start cmd /k "mongod"
timeout /t 3
start cmd /k "cd Module1_UserAuthentication\backend && npm start"
start cmd /k "cd Module2_ExperienceSharing\backend && npm start"
start cmd /k "cd Module3_PreparationGuidance\backend && npm start"
start cmd /k "cd Module4_SkillGapAnalysis\backend && npm start"
timeout /t 5
start cmd /k "cd IntegratedApp\frontend && npm start"
```

Then just run: `start-all.bat`
